# flow-skill

这是我的个人技能库，用来集中保存和迭代可复用的 AI 工作流 / 技能文件。

## 技能目录

| 技能 | 说明 |
| --- | --- |
| [PDF 首页高密度速览 + 证据型原文标注](./skills/pdf_evidence_literature_annotation_skill_v3/) | 对论文 PDF 生成首页速览卡，并在原文位置做少量、有重点的证据型标注。 |
| [PPT Master](./skills/ppt-master/) | 从 PDF、DOCX、URL、Markdown 或文本材料规划/生成可编辑 PPTX 的工作流入口；完整运行依赖上游工具链。 |

## 使用方式

每个技能单独放在 `skills/` 下的一个目录中，目录内至少包含：

- `SKILL.md`：技能主体说明
- `README.md`：技能简介、适用场景和版本要点

对于来自外部开源项目的技能，应尽量保留：

- 来源链接
- 原作者信息
- 许可证说明
- 是否为完整可运行版本
